# M1 数据基础 — 使用说明

## 已实现

- SQLite 数据库 `data/invest.db`（可通过 `DATABASE_URL` 换 PostgreSQL）
- 表：`instruments`, `ohlcv`, `macro_series`, `valuation_series`, `fund_nav`, `indicator_values`, `composite_scores`, `crawl_audit`
- 采集：
  - **crawl_us**：NDX(QQQ)、SPX(^GSPC)、VIX、FRED 美债10Y/美元指数
  - **crawl_jp_de**：N225、DAX、USD/JPY、EUR/USD、VDAX、美债10Y
- 校验：OHLC 合法性、最新 bar 价格跳变、provisional 不覆盖 official
- CLI：`python -m invest.jobs.daily_crawl`

## 命令

```bash
python -m invest.jobs.daily_crawl --init-db
python -m invest.jobs.daily_crawl --job crawl_us
python -m invest.jobs.daily_crawl --job crawl_jp_de
python -m invest.jobs.daily_crawl --job all
python -m invest.jobs.daily_crawl --health
```

## 环境变量

复制 `.env.example` → `.env`：

- `FRED_API_KEY`：采集 `macro:us10y`、`macro:dxy`（不配置则跳过并记入 partial）
- `DATABASE_URL`：默认 `sqlite:///./data/invest.db`
